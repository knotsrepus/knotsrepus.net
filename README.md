# knotsrepus.net
Source code for https://knotsrepus.net

## Getting Started

### Dependencies

- Hugo

### Installing

```shell
git clone https://github.com/knotsrepus/knotsrepus-api.git
cd knotsrepus-api/
./setup-hugo.sh
```

### Running

Windows:
```shell
./hugo.exe serve
```

macOS/Linux:
```shell
./hugo serve
```
